Kompetensi yang dievaluasi: Kemampuan memahami konsep prosedur dan fungsi.
Buatlah fungsi atau prosedur untuk menyelesaikan soal ini. Diberikan dua buah array of integer. Hitung rata-rata dari kedua array dan ambil hanya angka di depan koma dari masing-masing rata-rata. Lalu bandingkan, jika lebih besar rata-rata array pertama maka tampilkan rata-rata array pertama, jika lebih besar rata-rata array kedua, maka tampilkan rata-rata array kedua.
Format Masukan:

n1, 0 < n1 < 50, banyaknya jumlah elemen integer dari array pertama.
n1 baris integer isi array pertama.
n2, 0 < n2 < 50, banyaknya jumlah elemen integer dari array kedua.
n2 baris integer isi array kedua (tidak ada kasus rata-rata sama di server).

Format Keluaran:

rata-rata array pertama atau array kedua.

3
